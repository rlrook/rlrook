What is Duo?
Duo Security is a program for multi-factor authentication. It adds an additional layer of security to the information you access online.
How to Enable Duo for the First Time
Duo Security multi-factor authentication is available and required for all UMN students, staff, and faculty anywhere a UMN internet ID is used.
